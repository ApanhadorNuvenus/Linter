# Linter

Приложение для Android, позволяющее создавать базу карточек интервального повторения на основе любых импортированных текстов или видеороликов YouTube.
## Основной функционал:
  - Интерактивное чтение: перевод любого слова или словосочетания по нажатию или выделению
  - YouTube плеер: просмотр видео со встроенными кликабельными субтитрами
  - Интервальное повторение: повторение добавленных слов по упрощённому алгоритму FSRS

## Особенности:
- Офлайн-перевод (EN-RU, FR-RU): работает локально через Google ML Kit и модели Helsinki OPUS-MT (ONNX)
- Бесплатное использование, потому и перевод соответствущий. Частично нивелируется возможностью добавить свой перевод к любой карточке
